# v0.9.0-rc.1

## 版本定位

这是 `goweb-core` 的首个正式 release candidate，用于稳定运行时内核边界，不是 final stable。

## 本版本主要能力

- 基于 `net/http` 的轻量运行时与中间件
- PostgreSQL + GORM 初始化
- JWT 基础认证能力
- `permission_code` 导向的 RBAC 抽象
- 配置、日志、校验、错误、文件存储等基础设施

## 本版本不包含

- 业务模块
- 代码生成器
- 用户端 / admin 页面
- 队列、任务系统、WebSocket

## 兼容关系

- 兼容 `goweb-scaffold v0.9.0-rc.1`
- 目标兼容范围：`goweb-scaffold v0.9.x`

## 升级注意事项

- 这不是 `v1.0.0`
- 如果 scaffold 侧要进入真实业务试跑，先锁定 core/scaffold 版本对
